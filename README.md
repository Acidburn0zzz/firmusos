FirmusOS
========

Real-time Operating System 
Microkernel design
POSIX-compliant

========
Packages needed:

* GCC >= 4.8.1
* Kdevelop >= 4.6.0
* Grub2
* xorriso
* binutils
* QEMU (Ubuntu 13.10)
* Cmake
* ccache

Usage
========

In *kernel* directory type:

1. $: make clean
2. $: make

QEMU starts immediately.
